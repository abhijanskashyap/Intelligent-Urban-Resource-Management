Welcome to the Urban Resource Monitoring System

This application allows you to monitor, analyze, and predict urban resource consumption and traffic density. It also helps in detecting anomalies in waste levels and classifying events.

Features

  • Exploratory Data Analysis (EDA): View data information, descriptive statistics, missing values, and various visualizations.
  • Electricity Usage Prediction: Predict future electricity consumption based on various parameters.
  • Event/Anomaly Classification: Classify different events (e.g., Festival, Power Outage) based on resource usage patterns.
  • Waste Level Anomaly Detection: Identify unusual spikes or drops in waste levels.
